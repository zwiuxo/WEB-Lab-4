**⚙️ Как запустить проект**
1. **Установка зависимостей (Composer):**
Так как в папке `vendor` слишком много файлов, она не загружена в репозиторий. Перед запуском выполните команду в папке проекта:
```bash
docker run --rm -v ${PWD}/www:/app composer install
```
*(Или просто `composer install`, если он установлен на ПК)*.

2. **Запуск контейнеров:**
```bash
docker-compose up -d --build
```
3. **Ссылки для проверки:**
- Главная (Инфо и API): [http://localhost:8080/index.php](http://localhost:8080/index.php)
- Форма заказа такси: [http://localhost:8080/form.html](http://localhost:8080/form.html)
- Просмотр истории из файла: [http://localhost:8080/view.php](http://localhost:8080/view.php)

**📂 Содержимое проекта**
- `docker-compose.yml` — описание сервисов Nginx и PHP.
- `www/composer.json` — конфигурация библиотек.
- `www/ApiClient.php` — класс для запросов к API.
- `www/UserInfo.php` — класс для сбора информации о пользователе.
- `www/process.php` — процесс сохранения данных, работа с кешем и API.
- `www/index.php` — показ данных сессии, информации о юзере и API.
- `www/form.html` — HTML-форма заказа такси.

**🧠 Штрафное задание (Выполнено)**
